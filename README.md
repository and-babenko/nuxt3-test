5. Нужна мобильная разметка? -- Смены сетки пока будет достаточно.
6. Эффекты хувера при наведении на карточку? -- Только курсор.
7. Сохранять id картинки в URL, скажем, в качестве квери-параметра? -- Да (иначе смысл от копирования URL).
